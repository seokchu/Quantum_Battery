# 온디바이스 SoC 양자 디코더 논문 초안 (5페이지 이내, Double Column 용)

논문 제목: **QECCT 기반 양자 오류 정정을 위한 온디바이스 SoC 가속기 경량화 연구**

---

## Abstract (초록)
[시각화 자료: 없음]
양자 컴퓨팅의 실용화를 위해 양자 오류 정정(QEC) 연산은 큐비트 코히런스 시간 내에 실시간으로 이뤄져야 한다. 최근 Transformer 구조를 활용한 QECCT(Quantum Error Correction Code Transformer)가 기존 MWPM 디코더 대비 매우 우수한 논리 오류율(LER)을 달성함이 입증되었으나, O($Nd^2n$)의 계산 복잡도와 방대한 파라미터 크기 탓에 에지(Edge) SoC 및 온디바이스 환경 상의 하드웨어 전용 가속기에 포팅하기에는 무리가 있다. 본 연구에서는 Surface Code ($d=3$) 환경에서 Knowledge Distillation(지식 증류) 기반의 Student 모델(일반 레이어 $N=2$, 임베딩 차원 $d=32$)을 학습시켰다. 본 실험 결과 해당 모델은 원본 모델 파라미터의 약 2.5% 수준(약 39배 압축, 120KB)의 크기와 2.78배 빠른 추론 속도를 보임과 동시에 기존의 육중한 Teacher 모델에 극도로 가까운 수렴성(LER의 극히 미미한 손실율)을 유지하며 MWPM 알고리즘을 여전히 상회하는 성공적인 결과를 달성하였다. 추가로 구조적 프루닝(Structured Pruning)과 동적 INT8 양자화(Dynamic Quantization)를 통해 SoC 한계를 극복하기 위한 다단계 최적화 파이프라인의 Trade-off와 도메인 이식 가능성을 종합적으로 분석하였다.

## 1. Introduction (서론)
[시각화 자료: 향후 필요에 따라 표면 부호(Surface Code) 기본 형태 도식 추가 가능, 분량 조절 시 생략]
양자 컴퓨터는 양자 물리학적 중첩과 얽힘 현상을 활용하여 현존 고전 컴퓨터를 능가하는 병렬 처리 성과를 이룰 수 있다. 하지만 양자 비트(Qubit)의 정보는 열잡음 등 외부 환경에 취약하게 반응하며 파괴적 오류를 쉽게 발생시킨다. 이러한 양자 상태를 보존하기 위한 필수 과제로 표면 부호(Surface code) 등 위상 양자 오류 정정 부호 기반의 디코더 설계 기술이 핵심으로 연구되고 있다. 종전에는 Minimum Weight Perfect Matching(MWPM)이나 Union-Find와 같은 그래프 매칭 기반의 다항 시간 알고리즘이 널리 쓰였다. 그러나 대규모 물리적 큐비트의 확장(Scaling)과 코히런스 한계의 급박한 시간 제약(~μs)을 해결하기 위한 실시간 처리의 병목 문제로 인해, 점차 신경망(Deep Learning) 기반 디코더로 주도권이 넘어가고 있다.
최근 등장한 QECCT (Quantum Error Correction Code Transformer)는 패리티 체크 행렬(Parity check matrix)의 의존 관계 구조를 어텐션 마스크에 반영하여 MWPM을 훌쩍 능가하는 고도의 정밀도를 입증하였다. 
하지만 QECCT는 파라미터를 많이 소모하는 트랜스포머 기반의 아키텍처이기에, 극저온 시스템과 직접 연결되는 컨트롤러 내부의 제한적인 자원을 가지는 소형 칩(SoC, FPGA 등) 상단에 전개하기엔 연산량의 한계가 명확하다. 따라서 본 연구에서는 Knowledge Distillation(KD)를 주축으로 구조적 프루닝, INT8 타입 양자화가 결합된 압축 파이프라인을 제시하고 해당 파이프라인이 Teacher 모델 아키텍처의 탁월함을 어느 수준까지 유지하며 온디바이스로 침투할 수 있는지 검증한다.

## 2. Method (연구 방법론)
[시각화 자료: 블록 다이어그램 (KD를 거쳐 Pruning 및 Quantization으로 이어지는 방법론의 순서도 흐름) - 공간 여력이 될 경우 직접 그리기 좋음]

### 2.1 Teacher QECCT 아키텍처 제원
실험의 베이스라인이 되는 Teacher QECCT는 $d=3$ 표면 부호를 타겟으로 하며, 트랜스포머 레이어 수 $N=6$, 임베딩 차원 $d=128$, 다중 헤드 수 8로 구성되었다. 초기 노이즈 추정기(Noise Estimator), 어텐션 마스크 등 원형 논문에 따른 미분 가능 로직 구성을 동일하게 채택하여 200번의 완벽한 에폭(Epoch) 훈련을 거쳐 MWPM 디코더의 성능을 뛰어넘는 모델로 훈련시켰다.

### 2.2 Student QECCT 아키텍처 및 KD 학습 (Knowledge Distillation)
방대한 모델 크기를 SoC 수준으로 무너뜨리기 위해, 극소형 모델인 Student QECCT ($N=2$, $d=32$, 다중 헤드 수 4) 네트워크를 설계했다. 훈련에는 Task Loss (자체의 물리적 에러 추정 로스) 외에도 Teacher 측 Output Logit이 갖는 상관적 패턴 맵을 모방하는 KD Loss와 중간 Transformer Activation 형태를 1x1 선형 프로젝션으로 압축해 전달받는 Attention Transfer Loss를 선형 병합하여 최적화시켰다. 

### 2.3 구조적 프루닝 및 구조 양자화
학습된 Student 모델이 엣지 칩에 올라갈 때 희소 가속(Sparse Acceleration) 여부에 구애받지 않고 실제 행렬 연산량 자체를 보장받을 수 있도록 비구조적 방식이 아닌 L1-Norm 기반의 '구조적 통(Channel) 단위 프루닝(Structured Pruning, 비율 50%)'을 추가로 진행하였다. 마지막으로 FP32 형태의 파라미터를 메모리 대역폭이 비좁은 CPU, MCU에서의 직통 실행을 염두에 둔 INT8 Dynamic Quantization(동적 양자화)을 적용하여 용량을 최소 단위로 압축시켰다.

## 3. Experiment & Result (실험 및 결과)

이 섹션은 본 논문의 실질적 기여를 증명하는 부분이다. 표면 부호 거리($d=3$), 물리적 독립 오류율 $p \in [0.02, 0.14]$ 환경하에서 각 10,000번의 몬테카를로 독립 샘플링을 거쳐 검증되었다. 

### 3.1 디코더 논리 오류율 (LER) 성능
[시각화 자료: `fig_ler_comparison (1).png` (가장 핵심적인 꺾은선 그래프 Figure 1로 삽입 지시)]

학습이 불충분했던 기존 사전 테스트와 달리, 100만 스텝 레벨로 끝까지 수렴시킨 결과 Student 모델(KD)은 경이로운 회복력을 보여주었다. 물리적 오류율 $p=0.02$ 구간에서 Teacher의 0.0033의 LER을 거의 완벽히 따라잡은 0.0031 LER을 기록했으며, 노이즈가 강렬한 $p=0.10$ 수준에서도 Teacher(0.076)와 유사한 0.080을 기록했다. 특히 전 구간에서 고전적인 MWPM 최적화를 가볍게 따돌림으로써, 구조 축소를 진행하여 파라미터 수를 극단적으로 희생하더라도 KD의 지식 이식성이 양자 디코더 구조 도메인에서 강력히 유효함을 최초로 밝혀내었다.

### 3.2 모델 압축률 및 시스템 최적화 분석
[시각화 자료: `fig_model_size (1).png` (압축률 막대 비교 그래프 Figure 2로 삽입 지시)]

위 성능 유지에도 불구하고 모델의 경량화 폭은 실질적이었다. 1,193,084개에 달하던 Teacher 파라미터는 26,652개로, 약 2.5% 비율($\sim$39배)로 급속 감소하였다. 메모리 풋프린트는 4.69MB에서 순수 Student 구조상 120KB를 차지하였고, 이를 INT8 양자화 파일로 배포할 시 단 57KB를 나타냈다. 이는 매우 한정적인 SRAM을 보유한 초소형 칩에도 부담 없이 적재할 수 있는 수치이다. 추가적으로 FP32 기준 CPU 싱글 스레드 벤치마크 상에서 Teacher 모델 당 1.49ms 추론 시간을 요구했던 반면 프루닝 전 Student 모델 단일에서 0.54ms 수준으로 단축 시켜 최대 2.78배 성능 응답성을 제고했다.

## 4. Discussion (논의 및 한계)
[시각화 자료: 없음]
본 연구는 고전 디코더를 뛰어넘는 신경망 기반 양자 디코더를 모바일화시켰다는 강력한 이점을 지니지만 추가 제어 기법에서는 한계점을 동반했다. 
첫째, L1-Norm 기반으로 50%의 행렬을 차단한 프루닝 모델은, 모델이 원래 학습해야 하는 위상 부호 특성의 일부 로직 결손을 초래해 $p=0.10$ 구간에서 MWPM보다 더 열화된 성능 무너짐을 실측 데이터로 확인하였다. 
둘째, INT8 양자화 적용 시 파일 용량은 훌륭하게 줄었으나, PyTorch의 동적 양자화 로직 상 GPU의 병렬화 수혜가 단절되고 CPU 기반으로 순차처리 되면서 추론 지연시간이 FP32(Student) 모델보다 오히려 상승하는 프레임워크 한계 오버헤드 결과를 낳았다. 차후 SoC 환경 실리콘 보드 포팅 시 NPU 가속기와의 연성을 통해 속도 역전 현상이 해결 가능할 것으로 추정한다.

## 5. Conclusion (결론)
[시각화 자료: 없음]
본 연구는 깊고 무거운 트랜스포머 기반의 QECCT 디코더를 제한된 컴퓨팅 자원을 지닌 SoC 인터페이스 상단에 이식(On-Device)하기 위한 통합 경량화 방법론과 그 정량적 타당성에 대하여 논의하였다. 단순히 구조를 작은 N과 d로 잘라냈을 때 잃어버리는 막대한 성능 손실분을, Teacher-Student 간의 Knowledge Distillation(지식 증류) 로직으로 거의 온전히 복구할 수 있음을 실험적으로 입증하였다. 이를 통해 제안된 소형 디코더는 최소 파라미터 용량(120KB 내외)으로도 MWPM 성능을 상회할 수 있었으며 QEC(양자 오류 정정)의 병목을 풀 수 있는 경량급 초석으로 기여한다. 향후 연구로는 프루닝된 잉여 가중치들의 전이학습(Fine-tuning)을 통한 복구 여부 및 온전한 FPGA RTL 합성 및 레이턴시의 검증이 요구된다.
